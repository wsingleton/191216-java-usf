Remove spaces from string

Working with Strings is a huge part of being a developer. In fact, its probably the most used class in all of the Java language (don't quote me on that lol). This task serves as another opportunity to get familiar with the String API.

Your task is to simply remove all of the spaces from a provided string. There are no constraints associated with this task.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

Don't forget to cover cases where invalid arguments are provided!
