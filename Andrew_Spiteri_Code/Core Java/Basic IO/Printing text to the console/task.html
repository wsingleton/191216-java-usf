Printing text to the console

Welcome to your first Java coding lesson! This course and its lessons are created using the IntelliJ IDEA integrated development environment (IDE) along with the EduTools plugin.

Your first task is to implement a method which when invoked will print "Hello, world!" to the console.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

Try to get access to the output stream of the System class, on this member you will find a few useful methods which allow you to print text to the console.

Attention to detail is a vital skill for a developer! Be sure that your output matches exactly what was requested.
