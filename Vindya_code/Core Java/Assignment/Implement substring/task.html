Implement substring

Not that you'll be frequently implementing String API methods by hand, but it is common to be asked to implement them in coding challenges.

For this challenge: implement a substring method. The provided method signature takes in three arguments: the string from which substring shall be taken, the starting position of the substring, and the non-inclusive end of the substring.

The only constraint of this task is that you cannot use the existing substring method belonging to the String API.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

Don't forget to handle the base cases and common exceptions!
