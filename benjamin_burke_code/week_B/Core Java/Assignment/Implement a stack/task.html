Implement a stack

The stack is one of the most commonly used data structures in software development. This data structure follows the First In, First Out principal. This means that, much like a stack of plates, only the most recently added item is retrieved from the stack when using either the pop() or peek() methods of the instance.

Your task is to implement the logic behind the standard stack methods: push(), pop(), peek(), and size(). The stack should grow at a rate of one when elements are pushed into the stack when it is already at capacity; there is no need to decrease the size of the stack when elements are popped off of the structure.
