Buhlakify Reimbursement Policy




  • Food request must be less than $500 per request.
    • Must provide a receipt if $100.
    • Daily lunches do not count, unless it is client related.

  • Travel or Lodging request must be less than $500 per request.
    • Special cases will be considered if it exceeds standard limit.
    • Receipt must be provided.

  • Other requests must provide type in the description. The following are eligible:
    • Supplies
    • Training/development

  • ALL reimbursement requests must be within 60 days of the expense transaction.

  • ALL reimbursement requests must include a description to be considered for approval.

  • Any request that fails to comply with rules stated in this policy will be automatically denied.
