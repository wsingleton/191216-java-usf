What is CSS?

CSS allows you to create rules that specify how the content of an element should appear. For example, you can specify that the background of page is light grey and use the Arial typeface, or that all level one headings should appear in blue, italic, and Times New Roman font.

Understanding CSS Positioning: Thinking inside the box

One of the core concepts to understanding CSS is how it leverages the box model on our elements. CSS allows for us to create rules that control the way each section of the box model (content, padding, border, and margin) is presented.
