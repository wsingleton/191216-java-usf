Reading input from the console (with Scanner)

As you will quickly find out, in programming - like life - there are often times multiple ways to solve a single problem. The task of reading from the console is no exception to this. We saw in the last task how we could read from the console using the BufferedReader class, with a little help from the InputStreamReader class. Next, we will work on accomplishing the same task, but using the Scanner class.

Your task is the same as before, write an implementation which prompts the user from some input. This time, ask the user for some integer value using the Scanner class and return the provided value.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

You will notice that the Scanner class has a lot methods that could be used to read from the console. Be sure to choose the method that best fits the needs of this task.

Be sure that you are properly handling unchecked exceptions!
