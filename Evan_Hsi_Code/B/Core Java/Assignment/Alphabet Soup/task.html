Alphabet Soup

Techies love their TLAs - Three Letter Acronyms. In the next few weeks you will become acquainted with dozens (not exaggerating) of acronyms. Acronyms are simple mnemonics which help us to pack more information into a more smaller space. For instance, OOP stands for Object Oriented Programming - or Out Of Pocket, depending on whom you ask.

Your task is to take the provided string and make an acronyms out of the phrases (represented as Strings) provided to a method. Aside from the forbidden use of external libraries and frameworks, there are no other constraints associated with this task.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

If a word includes a hyphen, you should include the first letter of both parts of the compound word in the acronym.

The capitalization of letters or punctuation within the phrase doesn't matter!

Numbers should not be included in the returned acronym!

Remember to cover the base cases of null and empty strings!
