FizzBuzz

The classic FizzBuzz challenge! This is an extremely common coding challenge which is given to junior developers to ensure that they have basic problem solving and programming language skills before hiring them.

The task is to iterate across the provided array, and to populate - and return - an array of Strings based upon those values. If a number at a given index position is divisible by three, you should put the word "fizz" into the corresponding index position of the String array.

Likewise, if a number is divisible by five, you should put the word "buzz" into the String array. If a number is divisible by both three and five, the word "fizzbuzz" should be placed into the array instead.

Finally, if the number is not divisible by three nor five, you should simply put the String representation of the number into the String array.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

Don't forget to cover the base cases of null or empty arguments!
