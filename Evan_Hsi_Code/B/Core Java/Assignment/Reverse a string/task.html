Reverse a string

As with most common code challenges, there are various libraries at your disposal which can quickly and easily accomplish the task at hand. However, a code challenge is not meant to test your ability to use pre-made tools and libraries, but instead tests your ability to solve a problem using your implementation.

Your next task is probably one of the most commonly asked junior-level code challenges: reverse a string. Now, there are several variations of this question, each of which establish different constraints and allowances. For this task, you should reverse the provided string without using a temporary variable to store characters, nor should you use any native or third-party libraries to assist you with reversal (i.e. no using StringBuilder or StringBuffer!)

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.
