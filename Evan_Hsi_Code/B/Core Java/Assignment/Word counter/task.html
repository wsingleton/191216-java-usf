Word Counter

Extracting data from files for analysis is a common task for a junior developers. Your task is to implement two methods. One which will extract an array of Strings from a file, another which will provide a count of words from a provided array of Strings. The word count will be returned as as a Map of Strings and Integers.

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

Don't forget to handle base cases and common exceptions!

Remember, it is preferable to never return null from any method!
