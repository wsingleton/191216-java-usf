Iframes

An iframe is like a little window that has been place into your webpage. The term "iframe" is an abbreviation for "inline frame".

One common use of iframes is to embed Google Maps into the webpage. The content of the iframe can be to any HTML page (which can be located either externally or locally on our machine/web server)




Another common use of iframes is to embed videos into a page. Youtube provides excellent support for getting the iframe info regarding any video you wish to embed into your webpage.

Using iframes works just fine when we have an externally hosted video. However, if I have my own video content that is located on the web server as this document, then we will use the HTML5 <video> tag to accomplish this.
