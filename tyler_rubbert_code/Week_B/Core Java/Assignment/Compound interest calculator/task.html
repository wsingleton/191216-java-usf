Interest calculator

Your task is to implement a method which will return the appropriate return on investment (ROI) when provided with valid values - negative values are not permitted. The returned value should be rounded to two decimal places.

Compound interest is calculated using the following formula:
P (1 + R/n)^(nt)

Where,

  • P = initial principal amount.
  • R = interest rate
  • n = number of times interest applied per time period
  • t = number of time periods elapsed

Once you have implemented a solution, you can hit the Check button below to run your implementation against some unit tests provided by your trainer.

While there is no basic operator working with exponents, the java.lang.Math class provides some useful static methods to help you complete this task.

There are several different ways that you can round a double to a certain number of decimal places. Since there are no constraints on this task, feel free to use whatever implementation you desire.
